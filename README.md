# 🤖 AI QA Assistant

> Reduced test case creation time from 30 minutes → 30 seconds using AI

A professional AI-powered web application that automatically generates 
QA artifacts using Groq LLaMA AI. Built by Sarbani Mondal.

## 🔗 Live Demo
👉 https://ai-qa-assistant-b9el.onrender.com

## 🎯 Problem Statement
QA engineers spend 30+ minutes manually writing test cases for every module.
This tool generates 10 professional test cases in under 30 seconds using AI.

## ✅ Features
- 📋 AI Test Case Generator — generates 10 professional test cases instantly
- 🐛 AI Bug Report Generator — creates Jira-ready bug reports
- 📊 AI Test Data Generator — produces valid and invalid test data sets
- 📥 Excel Export — download test cases in .xlsx format
- 📄 PDF Download — professional PDF reports
- 🌙 Dark/Light Mode — modern responsive UI
